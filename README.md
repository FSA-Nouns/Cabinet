# Qabin

## What is Qabin?

> "A cozy place for your queries to live."

Qabin is a SQL Builder focus on intutive UI and seamless UX.

The user must uploads tables as .csv files and then can execute many types of query in his dataset.

## Technologies used

* Node.js
* Express.js
* React.js
* Redux.js
* Material-UI
